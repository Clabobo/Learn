## 具体增强项（分主题）
- Java 集合（docs/java/collections-internals.md）
  - HashMap：`hash` 分布、`resize`/`treeifyBin` 触发条件与成本、红黑树旋转简述、`fail-fast` 迭代器原理
  - LinkedHashMap：访问/插入顺序维护与 LRU 实战、阈值与命中率权衡
  - ConcurrentHashMap（补充对比）：分段/桶锁与 CAS、扩容协作
- 并发与线程池（docs/java/concurrency-threads-pools-locks.md）
  - Happens-Before 与内存屏障、`volatile` 有序性细节
  - ThreadPoolExecutor：`addWorker/runWorker` 生命周期、队列选择（SynchronousQueue/LinkedBlockingQueue/ArrayBlockingQueue）与拒绝策略语义
  - ReentrantLock：公平/非公平、条件队列、锁竞争热点治理
- JVM（docs/java/jvm-memory-gc-classloading.md）
  - GC：卡表/Remembered Set、分代与晋升、并发标记细节、暂停时间来源
  - 类加载：解析/链接/初始化流程、双亲委派的打破风险与隔离策略
- OOP 与设计模式（docs/java/oop-design-patterns.md）
  - SOLID 逐条配工程反例与重构方案
  - 模式内部消息流（策略/观察者/责任链），误用陷阱与性能代价
- Kotlin 互操作（docs/kotlin/kotlin-vs-java-interoperability.md）
  - `@JvmName/@JvmOverloads/@JvmStatic` 产物字节码展示、`DefaultImpls` 与 SAM 转换
  - 平台类型与空注解映射，互操作 NPE 溯源
- Kotlin 扩展/高阶（docs/kotlin/extensions-higher-order-functions.md）
  - `inline/noinline/crossinline/reified` 语义与二进制影响、闭包捕获成本
- Kotlin 空安全（docs/kotlin/null-safety.md）
  - 类型推断与智能转换局限、平台类型风险、契约（Contracts）与早失败模式

- Android 核心（进一步加深“原理”段）
  - Binder（docs/framework/binder-overview.md）：驱动队列、引用计数、事务标志（`ONEWAY` 等）、线程池上限与 SELinux 权限模型
  - Handler（docs/android-core/handler-looper-messagequeue.md）：`nativePollOnce` 与 epoll 等待、同步屏障调度细节
  - 绘制管线（docs/ui-view/view-drawing-pipeline.md）：RenderThread → Skia 提交流程、脏区与合成
  - 事件分发（docs/ui-view/touch-event-dispatch.md）：`TouchTarget` 链管理、pointer id 多指序列
  - ART（docs/art-vm/*）：ClassLinker 解析与镜像关联、Profile 文件结构更细字段说明
